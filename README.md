## 📋 Descripción

Proyecto de práctica desarrollado con **JavaScript puro** que simula un carrito de compras funcional. Permite a los usuarios agregar productos, gestionar cantidades, eliminar artículos individualmente y vaciar el carrito por completo.

Este proyecto forma parte de mi aprendizaje en **JavaScript moderno**, enfocándome en manipulación del DOM, eventos y manejo de estados.

## ✨ Funcionalidades Actuales

- ✅ Agregar cursos al carrito de compras
- ✅ Eliminar cursos individualmente
- ✅ Vaciar carrito completo
- ✅ Actualización dinámica de cantidades (productos duplicados)
- ✅ Interfaz responsive y amigable
- ✅ Visualización de productos con imagen, título, precio y cantidad
